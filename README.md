## Descripción del proyecto

Este trabajo práctico fue realizado utilizando GitHub y Jira para aplicar conceptos de control de versiones, gestión de tareas y trabajo colaborativo.

Se desarrolló un pequeño análisis sobre un conjunto de datos de ventas con el objetivo de obtener información básica sobre el desempeño de las ventas registradas.

## Estructura

* datos: contiene el archivo de ventas utilizado para el análisis.
* scripts: contiene el código desarrollado en Python.
* resultados: destinada a almacenar los resultados obtenidos.

## Herramientas utilizadas

* GitHub
* Jira
* Python
